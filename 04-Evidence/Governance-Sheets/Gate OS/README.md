Purpose

Documents the Google Sheets governance layer supporting the HyperSync Gate Operating System.

This package provides governance over authorization decisions, replay execution, dead-letter handling, operational alerts, and recovery controls.

Google Sheets remain the operational system-of-record while GitHub serves as the governance documentation and evidence repository.
