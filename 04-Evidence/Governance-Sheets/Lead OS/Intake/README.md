Purpose

Documents the Google Sheets governance layer supporting the HyperSync Lead Intake Operating System.

These sheets serve as the operational system-of-record for intake execution, event tracking, external lead ingestion, staging management, and intake-to-processing handoff governance.

Google Sheets remain the authoritative governance layer while GitHub serves as the documentation and evidence repository.
