Purpose

Documents the Google Sheets governance layer supporting the HyperSync Lead Process Operating System.

This governance package tracks sanitization, validation, routing, CRM preparation, campaign triggering, replay operations, and audit controls occurring between Intake OS and Outbound OS.

Google Sheets remain the operational system-of-record while GitHub serves as the governance documentation and evidence repository.
