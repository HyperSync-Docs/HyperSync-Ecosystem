Purpose

Documents the Google Sheets governance layer supporting the HyperSync Lead System Operating System.

This package provides audit governance, alert governance, replay governance, state governance, and operational oversight across all Lead OS conveyor execution activities.

Google Sheets remain the operational system-of-record while GitHub serves as the governance documentation and evidence repository.
