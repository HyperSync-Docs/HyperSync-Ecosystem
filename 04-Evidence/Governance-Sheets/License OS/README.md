Purpose

Documents the Google Sheets governance layer supporting the HyperSync License Operating System.

This package provides governance over license provisioning, entitlement management, billing lifecycle activity, webhook registration, audit controls, and replay recovery.

Google Sheets remain the operational system-of-record while GitHub serves as the governance documentation and evidence repository.
