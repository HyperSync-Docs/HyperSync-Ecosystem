SYS-LIC-GATE-Z1 — License Validation (POST)

SYS-LIC-GATE-Z1 serves as the authorization gateway of the Gate Operating System.

The workflow receives validation requests, retrieves license records, evaluates authorization conditions, determines execution eligibility, and records authorization outcomes.

Responsibilities include:

License validation requests
Registry lookup
Active status validation
Webhook key validation
Volume enforcement
Expiration enforcement
Suspension enforcement
Authorization response generation

This workflow determines whether execution is authorized or denied.
