Overview

Lend-PRC-Z2 — Validate DTI, Credit Range, Income Docs

Lend-PRC-Z2 serves as the underwriting validation stage of the Lender Process Operating System.

The workflow monitors the authoritative Process State Spine, validates conveyor readiness, acquires processing ownership, executes lending validation logic, persists validation results, and advances the application into routing decision processing.

Validation responsibilities include:

DTI calculation
Credit score validation
Income documentation validation
Pass / Fail / Incomplete determination
Validation evidence generation
State progression management
