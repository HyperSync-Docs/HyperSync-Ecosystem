SYS-LIC-Z2 — Lifecycle / Mutation Controller

SYS-LIC-Z2 serves as the license mutation workflow of the License Operating System.

The workflow receives lifecycle events, normalizes payloads, evaluates mutation logic, updates licensing records, manages webhook registrations, and performs bidirectional acknowledgements.

Responsibilities include:

Lifecycle event processing
Status calculation
Registry updates
Registry creation
Event logging
Webhook registration management
Acknowledgement processing

This workflow governs all operational license state transitions.
