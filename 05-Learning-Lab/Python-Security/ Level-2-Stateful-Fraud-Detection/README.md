Purpose

Documents Level 2 of the HyperSync Python Security Learning Path.

This learning package introduces stateful fraud detection systems and teaches how to track behavior across time, users, transactions, and correlated activity patterns.

The lessons focus on behavioral monitoring, rolling aggregation, anomaly detection, correlation analysis, and fraud prevention techniques used throughout modern security platforms.

Unlike Level 1, which validates individual events, Level 2 teaches how to evaluate patterns that emerge over multiple events and multiple identities.

Learning Objectives

Upon completion of Level 2, the learner will understand:

Per-User Stateful Aggregation

Rolling Window Aggregation

Multi-Metric Detection

Sequence + Aggregation Enforcement

Idempotent Aggregation

Metric Synchronization

Baseline Anomaly Detection

Controlled Drift

Cross-User Correlation Detection

Golden Fraud Detection Pipelines
HyperSync Alignment

Level 2 directly supports future HyperSync security architecture including:

Bot Detection

Behavior Monitoring

Velocity Analysis

Replay Detection

Fraud Detection

Risk Scoring

Compliance Monitoring

AI Security Workflows
