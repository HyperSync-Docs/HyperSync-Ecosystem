Purpose

Documents Level 3 of the HyperSync Python Security Learning Path.

This learning package introduces streaming joins and cross-stream fraud detection systems. The lessons focus on validating multiple event streams, joining related events by key, enforcing time-windowed joins, handling out-of-order arrivals, preventing duplicate joins, aggregating cross-stream behavior, and detecting fraud through relationships across users, devices, IPs, logins, and transactions.

Level 3 advances beyond Level 2 stateful fraud detection by teaching how fraud can be detected through timing, context, and relationships across multiple data flows.

Learning Objectives

Upon completion of Level 3, the learner will understand:

Multi-Stream Input Validation

Event Joining by Key

Time-Windowed Joins

Out-of-Order Join Handling

Join Idempotency

Multi-Stream Aggregation

Cross-Stream Anomaly Detection

Cross-Stream Drift Detection

Multi-Entity Correlation
